# Live Weather Tracker 🌦️

A simple, responsive, and beautifully designed Weather API application built for the CODTECH Internship. 

## Features
- **Live Data**: Fetches real-time temperature, humidity, and weather conditions.
- **No API Key Required**: Uses the free Open-Meteo API for beginner-friendly setup.
- **Indian English Terminology**: Provides weather summaries in localized language (e.g., "Today’s Weather in Delhi is Sunny with 32°C").
- **Premium UI**: Modern glassmorphic design that works perfectly on both desktop and mobile devices.

## How to Run
Since this is a front-end only application, no installation or server is required!
1. Clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. Search for any city to see the live weather update!
